# **Dependencies**
```
sway, wofi, mako, adw-gtk-theme, swaybg, grim, kitty, thunar, ttf-iosevka-nerd, xdg-desktop-portal, xdg-desktop-portal-wlr #screen casting
```

# **Installation**
```
git clone https://github.com/wirosmango/sway-dotfiles.git
cd sway-dotfiles
cp -r * ~/
```

**Done!**
